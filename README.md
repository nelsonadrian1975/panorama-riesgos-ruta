# Panorama de Riesgos de Ruta

Aplicacion local en HTML, CSS y JavaScript para elaborar panoramas de riesgos de ruta, registrar puntos criticos, gestionar viajes de convoy, anexos, personal involucrado y generar una salida imprimible en PDF.

## Uso

Abra `index.html` en el navegador o use `abrir_panorama.bat` en Windows.

La aplicacion permite guardar y abrir proyectos editables `.panr.json`. El PDF se genera desde la seccion `PDF` usando la impresion del navegador.

## Estructura

- `index.html`: interfaz principal.
- `styles.css`: estilos de pantalla e impresion.
- `app.js`: cargador de modulos JavaScript.
- `src/app-*.js`: logica completa de la aplicacion.
- `dist/DISTRIBUIBLE_USB/LEEME.txt`: instrucciones del distribuible generado localmente.
